快捷访问 ​

通过快捷访问功能，您可以像使用本地应用一样快速访问本站。只需将本站添加到主屏幕或桌面，您便能像点击 App 图标一样轻松进入，无需每次输入网址。

iOS 操作步骤 ​

注意

仅支持系统自带的 Safari 浏览器

  1. 打开浏览器访问本站
  2. 点击 分享按钮
  3. 选择 添加到主屏幕
  4. 点击 添加

Android 操作步骤 ​

注意

仅支持 Chrome / Edge 等支持 PWA 的现代浏览器

  1. 打开浏览器访问本站
  2. 点击浏览器右上角的菜单（三点按钮）
  3. 选择 添加到主屏幕 或 安装应用
  4. 点击 添加

Windows 桌面端操作步骤 ​

注意

仅支持 Chrome / Edge 等支持 PWA 的现代浏览器

  1. 打开浏览器访问本站
  2. 点击地址栏右侧的 安装按钮（PWA 提示）
  3. 点击 安装
  4. 网站会生成桌面快捷方式或独立窗口，像应用一样打开

Mac 桌面端操作步骤 ​

注意

仅支持 Chrome / Edge / Safari（部分 Safari 版本支持 PWA）

  1. 打开浏览器访问本站
  2. Chrome / Edge：点击地址栏右侧的 安装按钮（PWA 提示）
    Safari：点击菜单栏 文件 -> 添加到主屏幕
  3. 点击 安装 或 添加
  4. 网站会生成独立窗口或桌面快捷方式，像应用一样打开

Linux 桌面端操作步骤 ​

注意

仅支持 Chrome / Edge 等支持 PWA 的现代浏览器

  1. 打开浏览器访问本站
  2. 点击地址栏右侧或浏览器菜单中的 安装按钮 或 添加到桌面
  3. 点击 安装 或 添加
  4. 网站会生成桌面快捷方式或独立窗口，像应用一样打开
